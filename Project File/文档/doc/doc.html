网站策划文档
匿名用户
  • 匿名操作
    1. 搜索
    2. 点评店铺
    3. 下载店铺信息
普通用户
  • 填写注册信息
    1. 用户名（全站唯一的用户名）
    2. 密码
    3. 邮箱地址（全站唯一的邮箱地址，用于找回密码或其它功能使用）
  • 用户操作
    1. 搜索
    2. 点评店铺
    3. 订位
      描述：查看历史订位信息、删除（只是逻辑上的删除，不是物理上的删除，方便总系统管理订位信息）
    4. 发邮件邀请朋友们在某个酒店聚餐
      描述：查看历史邀请聚餐信息，删除（同订位）
    5. 收藏店铺
    6. 下载店铺信息
    7. 会员积分（积分规则待定…）
    8. 帐户设置
      1. 更改邮件地址
      2. 更改登录密码（需登录，并且输入正确的原始密码，才能设置新的密码）
    9. 找回密码（系统发送更改新密码的链接地址，通过此地址可以设置新的登录密码）
店铺用户（上海、成都、北京…）
  • 填写注册信息
    1. 用户名（城市唯一的用户名）
    2. 密码
    3. 邮箱地址（城市唯一的邮箱地址，用于找回密码或其它功能使用）
    4. 店铺名称（可重复，由店铺地址来区分相同名称的店铺）
    5. 店铺所在城市（系统自动显示当前注册用户访问的城市）
    6. 店铺所在地区、地点（二级联动选择）
    7. 店铺详细地址
    8. 电话
    9. 手机
    10. 联系人
  • 店铺用户操作
    1. 基本信息修改
      描述：未审核的店铺用户信息将不会被系统搜索，审核后将无法修改 “ 红色 ” 项
      1. 店铺名称
      2. 店铺所在地区、地点
      3. 店铺详细地址
      4. 交通路线
      5. 电话
      6. 手机
      7. 联系人
    2. 店铺参数修改
      1. 包间
      2. 星级
      3. 人均消费（下拉选择框 如:49-99，99-149）
      4. 营业时间
      5. 总座位数
      6. 信用卡
      7. 停车场
      8. 设备服务
      9. 备注
    3. 店铺介绍
    4. 店铺公告（最新活动信息或其它内容）
    5. 店铺照片（最多10张）
    6. 菜单
      1. 添加菜
        1. 分类（由系统统一指定菜系分类）
        2. 菜名
        3. 图片（仅一张，系统生成大图与小图）
        4. 价格
      2. 删除菜（并且要删除菜的图片）
      3. 菜排序
      4. 特色菜/推荐菜/招牌菜（从菜单中设置）
    7. 地图（在地图上标注自己所在位置）
    8. 查看订位
    9. 查看邀请
    10. 帐户设置
      1. 更改邮件地址
      2. 更改登录密码（需登录，并且输入正确的原始密码，才能设置新的密码）
    11. 找回密码（系统发送更改新密码的链接地址，通过此地址可以设置新的登录密码）
店铺展示
  • 导航（店铺首页、详细信息、店铺照片、菜单、地图、订位、聚会邀请、用户点评）
  • 店铺首页
    描述：店铺介绍、店铺公告、特色菜（在首页展示的栏目待定）
  • 详细信息
    描述：店铺基本信息与参数
  • 店铺照片
    描述：店铺照片（小图列表，点小图看大图）
  • 菜单
    描述：将店铺所有的菜列表出来,可按菜单分类查看（分类、菜名、图片、价格）
  • 地图
    描述：利用地图接口在地图上标注店铺所在位置
  • 订位（普通用户）
    描述：普通用户填写订位表单提交到总系统，由总系统管理人员以人工方式通知店铺管理人员与普通用户确认订位
  • 聚会邀请（普通用户）
    描述：普通用户填写邀请表单提交到总系统，由系统自动将邀请信息和店铺介绍发送到被邀请的用户邮箱中
  • 用户点评（匿名用户、普通用户）
    1. 点评选项（服务好、质量好、环境好、性价比高）
    2. 人均消费（？元/人）
    3. 点评内容（纯文本）
  • 收藏店铺（普通用户）
  • 下载店铺信息（匿名用户、普通用户）
  • 人气（按店铺的访问次数来计算）
